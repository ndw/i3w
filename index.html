Interactivity Three Ways

This website supports the paper, Interactivity Three Ways by Norman Walsh and Michael Sperberg-McQueen, originally published in the proceedings of Balisage: The Markup Conference 2021. The paper was presented on Sunday, 1 Aug 2021.

The original source files and the build script are on GitHub.

The details of the project can be found in the paper. Three demonstrations of the “interactive schedule-at-a-glance” are presented here:

  • Using “plain old JavaScript”,

  • Using Saxon-JS, and

  • Using XForms.
